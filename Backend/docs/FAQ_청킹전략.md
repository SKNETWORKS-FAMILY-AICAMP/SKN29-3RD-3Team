## 3. 이미지 처리

### 조사 결과

총 1,892개 이미지가 포함되어 있으나 **내용과 무관한 디자인 요소**가 대부분입니다.

| 이미지 종류 | 개수 | 처리 방법 |
|-----------|------|---------|
| 헤더 로고 (주택청약 FAQ) | 전 페이지 | 무시 |
| 섹션 구분 배너 | 전 페이지 | 무시 |
| 타임라인 다이어그램 | 5개 페이지 (p42~46) | **무시** |

| 건강보험 구조도 | 1개 페이지 (p180) | **무시** |

### 무시해도 되는 이유

- 타임라인 다이어그램(p42~46): 핵심 조건(연속 90일, 연간 183일)이 본문 텍스트에 이미 포함됨
- 세대원 가계도(p62): 답변 텍스트에 "C(본인), B(배우자), A(배우자의 직계존속)..." 형태로 설명됨
- 건강보험 구조도(p180): 구조도 내 레이블이 본문에 함께 서술됨

## 4. 청킹 전략

### 청크 단위: Q&A 쌍

질문과 답변을 하나의 청크로 묶습니다.

```
[청크]
Q8. 해외체류 기간 산정 방법

「주택공급에 관한 규칙」 제4조제5항 및 제34조에 의하면
수도권의 투기과열지구에서 공급되는 주택을 우선공급
받기 위해서는 해당지역에 2년 이상 거주하고 있어야 하며...
```

**이렇게 하는 이유**: 사용자 질문이 자연어이고, FAQ의 질문 텍스트도 자연어이므로 embedding 유사도가 높게 측정됩니다. 답변만 청크로 만들면 질문과의 매칭이 약해집니다.

### 가. 주요내용 섹션 처리

각 중분류 앞에 붙는 법령 조문 해설 섹션("가. 주요내용")은 Q번호가 없어 Q&A 청킹에서 자동으로 제외됩니다. 이 내용은 이미 법령 원문 청커(LawChunker)가 별도로 처리하므로 중복 저장을 피할 수 있습니다.

## 5. 전처리 흐름

```
PDF 텍스트 추출 (pdftotext)
        ↓
페이지 분리 (폼피드 \f 기준)
        ↓
목차 페이지 제거 (점선 ·····이 3줄 이상인 페이지)
        ↓
헤더 제거 ("주택청약 FAQ", "www.molit.go.kr" 등)
        ↓
Q번호 판별 (단독 숫자 줄 + 다음 줄이 질문 텍스트)
        ↓
노이즈 제거 (법령 조항 번호, 대분류 헤더로 이어지는 경우)
        ↓
Q&A 청크 생성
        ↓
ChromaDB 저장
```

### Q번호 판별 기준 (노이즈 제거)

단독 숫자 줄이라도 다음 줄이 아래에 해당하면 Q번호가 아닌 것으로 처리합니다.

| 다음 줄 패턴 | 판별 결과 | 예시 |
|-----------|---------|------|
| `3. 제1호 및 제2호에도 불구하고...` | ❌ 법령 조항 번호 | 주요내용 섹션 |
| `Ⅰ. 청약자격(공통)` | ❌ 대분류 헤더 | 페이지 넘김 후 반복 |
| `해외체류 기간 산정 방법` | ✅ Q번호 | 실제 FAQ |
| `경기도 과천시에서 공급되는 주택의...?` | ✅ Q번호 | 실제 FAQ |
